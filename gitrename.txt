Базовые гид команды:
git init - инициал. локального репоз.
git add { либо имя ффайла либо поставить просто точку} - добавление файла в индексанцию  
git rm filename --cached - удаляет файл из индексации
git commit -m "сообщение"
git log - просмотреть commit, который есть
git branch просмотр всех веток
git barnch <name> - создание новой ветки
git checkout <имя ветки> - подключиться на другую ветку 
git merge имя ветки - вливание в текущую ветку указанной ветки
git branch -d <имя ветки> - удаление ветки
git remote - показать все удаленные репозитории
git remote add origin <ссылки>
git push origin main - отправляет изменения из локального в удаленный репозиторий, где origin (имя удаленного репозитория), а main (удаленная ветка) 
git clone <ссылка на репозиторий> - кланировать репозиторий на пк
